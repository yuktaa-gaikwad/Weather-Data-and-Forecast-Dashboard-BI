# 🌦 Weather Data & Forecast Dashboard

## 📌 Overview
This Power BI dashboard visualizes **real-time weather data** and **forecasts** using the [WeatherAPI.com](https://www.weatherapi.com/) service.  
It provides insights such as:
- Current weather conditions
- Hourly & daily forecasts
- Key weather metrics (temperature, humidity, wind speed, etc.)

---

## ⚙ Features
- **Real-Time Weather**: View the latest weather updates.
- **Forecast**: Get 7-day weather predictions.
- **Interactive Filters**: Explore data by location and time.
- **Visual Insights**: Charts and KPIs for quick understanding.

---

## 🛠 Tools Used
- **Power BI Desktop** – for building and viewing the dashboard
- **WeatherAPI.com** – for fetching weather data

---

## 🚀 How to Use
1. **Download & Install** Power BI Desktop  
   [Download Here](https://powerbi.microsoft.com/desktop/)
2. Open the file:  
   `weather.pbix`
3. Click **Refresh** to load the latest weather data (requires WeatherAPI setup in Power BI).

---

## 📜 Data Source
Weather data is fetched from **WeatherAPI.com**.  
Sign up for a free API key here: [https://www.weatherapi.com/signup.aspx](https://www.weatherapi.com/signup.aspx)
